Real-Time Roller Coaster Braking System

This website showcases my Application 6 project, where I built a real-time embedded braking controller for a fictional Orlando-based theme-park engineering company called Orlando ThrillWorks. The system runs on an ESP32 using FreeRTOS and demonstrates deterministic braking, emergency-stop handling, timing logs, and multiple concurrent tasks running under real-time constraints.

What This Prototype Demonstrates

  • Hard real-time tasks running every 20 ms
  • Instant emergency stop via ISR + binary semaphore
  • Queue, mutex, and critical section synchronization
  • Soft tasks for telemetry and heartbeat indicators
  • Timestamps proving hard deadline consistency
