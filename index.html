Real-Time Roller Coaster Braking System

A timing-critical braking controller prototype for the fictional Orlando ThrillWorks theme-park engineering team.

This website showcases my Application 6 project, where I built a real-time embedded braking controller for a fictional Orlando-based theme-park engineering company called Orlando ThrillWorks. The system runs on an ESP32 using FreeRTOS and demonstrates deterministic braking, emergency-stop handling, timing logs, and multiple concurrent tasks running under real-time constraints.

What This Prototype Demonstrates

  • Hard real-time tasks running every 20 ms for sensor sampling and brake control.
  • Instant emergency stop via ISR + binary semaphore that latches the brakes on.
  • Queue, mutex, and critical section synchronization between concurrent tasks.
  • Soft tasks for telemetry logging and a heartbeat LED for system status.
  • Serial timestamps that prove hard deadlines are consistently met.
